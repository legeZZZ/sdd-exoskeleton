# SDD Brownfield Project Onboarding

Onboard an existing project to SDD exoskeleton.

- [ ] Detect project structure (`sdd init --dry-run`)
- [ ] Review detected languages and structure
- [ ] Index with CodeGraph
- [ ] Generate initial specs
- [ ] Review and refine auto-generated specs
- [ ] Set up Obsidian vault
- [ ] Configure sync strategy (manual | git-hook | watch)
- [ ] First sync (`sdd sync`)
