# SDD Full Cycle Workflow

Complete SDD development cycle from initialization to archive.

- [ ] Initialize SDD (`sdd init`)
- [ ] Analyze architecture (`sdd analyze`)
- [ ] Propose feature/change (write proposal.md)
- [ ] Review proposal (team review)
- [ ] Implement change (write code + tests)
- [ ] Sync specs (`sdd sync`)
- [ ] Verify (`sdd doctor`, `sdd status`)
- [ ] Archive completed change
